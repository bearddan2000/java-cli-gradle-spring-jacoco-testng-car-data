# java-cli-gradle-spring-jacoco-testng-car-data

## Description
A POC for spring app. Testing done with jacoco
plugin using TestNG.

## Tech stack
- java
- gradle
  - spring
  - jacoco
  - testng

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
